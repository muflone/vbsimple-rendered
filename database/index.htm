Database

La sezione è dedicata all'utilizzo di Database di varia natura con Visual Basic, tramite una delle tante interfacce di accesso ai dati e l'uso del linguaggio SQL.
Dove non esplicitamente specificato, tutti gli esempi fanno riferimento a Database di tipo Access o Microsoft SQL Server.

Purtroppo l'argomento Database è davvero immenso ed oltre il 50% dei problemi rilevati riguardano quest'argomento; non potranno quindi essere trattate tutte le problematiche possibili. Si raccomanda pertanto la consultazione del Forum dedicato per una più ampia possibilità.
