Chat di VBSimple

Utilizzo della chat | Aiuto | Entra in chat

La chat di VBSimple è basata su un'applet Java e pertanto richiede la corretta installazione della Java Virtual Machine (JVM) ed una serie di autorizzazioni per poter collegare il computer dell'utente con il server centrale della rete IRC AzzurraNET (irc.azzurranet.org).

Questa chat infatti si interfaccia con una rete IRC italiana di nome AzzurraNET. Le caratteristiche, le opzioni e le regole sono quelle classiche di una rete IRC (Internet Relay Chat). La rete si compone di tanti canali (le stanze, solitamente tematiche) in cui è possibile chiacchierare, porgere domande o discutere delle più disparate argomentazioni. Il servizio è totalmente gratuito e non richiede connessioni secondarie ma utilizza la connessione Internet attualmente attiva.

Installazione della chat

L'applet Java per poter effettuare la connessione al server remoto irc.azzurranet.org necessita che venga installato il modulo di identificazione. Prima di ogni utilizzo della chat sarà quindi presentato un avviso di protezione come quello mostrato nella Figura 1.


Figura 1

Si tratta fondamentalmente di concedere al browser il permesso di stabilire una connessione al server utilizzando l'applet. Per i meno fidati è disponibile il codice sorgente del programma completo all'indirizzo http://www.mandragor.org/mandraforge/download.php?id=2 con annessa licenza GPL che ne consente anche la modifica e la redistribuzione.

Caratteristiche della chat

Quest'applet Java riproduce quasi tutte le caratteristiche di un più complesso client IRC, in grado di gestire oltre 10 canali contemporanei, finestre private (query), avvisi (notices), riconoscere gli indirizzi internet e trasformarli in collegamenti cliccabili.

Sono supportati anche i colori e le formattazioni come grassetto, sottolineato ed invertito.

Regole di AzzurraNET

Come tutte le comunità al mondo, anche AzzurraNET ha le sue regole, necessarie per la normale convivenza. Le regole non sono troppo limitative e si limitano fondalmentelmente a proibire le attività illegali e le cose non gradite.

È proibito l'accesso al server mediante Proxy, l'uso eccessivo di cloni e lo spam.

Ogni singolo canale inoltre stabilisce le sue regole aggiuntive: ad esempio alcuni canali sono moderati e possono parlare soltanto coloro che ne ricevono il diritti, altri canali proibiscono che si disturbino gli operatori (@), etc..

Durante la fase di collegamento al server apparirà nella finestra Stato il Messaggio del giorno (MOTD - Message Of The Day) che spiega a grandi linee le regole di comportamento.
