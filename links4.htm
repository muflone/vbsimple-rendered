Siti riguardanti altre notizie

  • Microsoft Data Access
    http://www.microsoft.com/data

    La sezione di casa Microsoft dedicata al Data Access, ovvero alle connessioni a database tramite ADO, RDS e UDL. Sono presenti anche gli MDAC in versone Runtime (RTM) e Source Development Kit (SDK) per lo sviluppo di applicazioni legate ai database.

  • Wotsit
    http://www.wotsit.org

    Un completissimo sito contenente le specifiche tecniche dei formati di file più comuni. Sono spiegati tantissimi tracciati di formati di files ordinati per tipologia ed estensione. Per ogni formato sono presenti varie documentazioni.

  • AllApi
    http://www.allapi.net

    Una completissima guida sull'API di Windows da consultare sia online che offline. È possibile infatti scaricare l'API Guide, un interessantissimo programma con centinaia di esempi sull'utilizzo dell'API.

  • Faqs.org
    http://www.faqs.org

    Il sito dedicato alle FAQs (Frequently Asked Questions) su tanti argomenti, newsgroup o altro genere di informazione.
    Davvero interessante anche la sezione dedicata alle RFC (Requests For Comments) sui protocolli Internet (http://www.faqs.org/rfcs).

  • RFC.net
    http://www.rfc.net

    Il sito ufficiale sulle RFC (Requests For Comments) sui protocolli Internet.

  • The Napster Protocol
    http://f11.org/david.weekly.org/napster.html

    Una guida al funzionamento del protocollo di trasferimento di Napster.

  • ADOX FAQ
    http://www.able-consulting.com/ADOX_Faq.htm

    Le FAQ (Frequently Asked Questions) su ADOX ovvero ADO eXtension, la libreria per la gestione delle strutture dati di ADO (ActiveX Data Objects).
    Il sito (http://www.able-consulting.com) contiene tante altre interessanti FAQ ed articoli dedicati alla programmazione.

  • Comunicazioni con le porte seriali
    http://www.maronese.com/mes/Articoli/Mscomm/mscomm.htm

    Un tutorial sul funzionamento ed utilizzo delle porte seriali. Se l'indirizzo non dovesse funzionare è presente una copia del tutorial all'indirizzo http://www.it-lang-vb.net/download.asp?file= Documentazione/Articoli/MSCOMM.htm&ID=362.

  • DLL Help Database
    http://support.microsoft.com/servicedesks/fileversion/dllinfo.asp

    Una sezione del sito Microsoft ignota a molti. Contiene un archivio informativo sulle DLL fornite da Microsoft con la possibilità di effettuare ricerche.

  • Platform SDK Update
    http://www.microsoft.com/msdownload/platformsdk/sdkupdate/

    Il platform SDK ufficiale. La fonte di documentazione delle tecnologie Microsoft più completa al mondo. Sono presenti vari SDK da scaricare per la consultazione offline.
    Un must-have per tutti gli sviluppatori di software a livello professionale.

  • TX Text Control
    http://www.textcontrol.com

    Un potente componente di programmazione che permette agli sviluppatori di aggiungere facilmente nelle loro applicazioni sofisticate caratteristiche di formattazione e visualizzazione del testo, viste solitamente solo in importanti programmi di elaborazione testi. È possibile scaricare la versione di prova con tutte le caratteristiche.
