  • VBP (Visual Basic Project)
    Files di progetto: la lista dei files coinvolti nello sviluppo di un programma.

  • VBG (Visual Basic Group)
    Gruppo di progetti: elenco di progetti connessi tra loro per lo sviluppo del programma.

  • VBW (Visual Basic Windows)
    Lista delle finestre aperte e loro posizioni all'interno di un progetto.

  • FRM (Form)
    Form (finestra) per effettuare l'interazione con l'utente.

  • FRX (Binary Form)
    Dati binari di un form: le icone, i puntatori e le immagini di un form.

  • CLS (Class)
    Modulo di classe contenente istruzioni e classi da istanziare per la loro riutilizzazione.

  • BAS (Basic Module)
    Modulo contenente istruzioni e funzioni da non istanziare, tipicamente pubbliche.

  • CTL (User Control)
    Controllo Utente da riutilizzare all'interno di un progetto, del tutto simile ai controlli Thunder.

  • CTX (Binary User Control)
    Dati binari di un controllo utente: icone, puntatori e immagini.

  • PAG (Property Page)
    Pagina delle proprietà di un controllo utente, di supporto per l'impostazione delle proprietà d'esso.

  • PGX (Binary Property Page)
    Dati binari di una pagina di proprietà: le sue icone, puntatori e le immagini.

  • DOB (User Document)
    Documento utente da utilizzare in applicazioni che devono girare in una rete, come ad esempio Internet.

  • DOX (Binary User Document)
    Dati binari di un documento utente: icone, puntatori ed immagini.

  • RC (Resources to Compile)
    File di risorse da compilare. Elenco delle stringhe da inserire nel progetto e dei files binari da includere.

  • RES (Resources Compiled)
    File di risorse compilato, pronto ad essere inserito in un progetto.

  • DEP (Dependencies)
    Elenco delle dipendenze di un progetto compilato. Tutti i files all'interno d'esso dovranno essere installati nel computer in cui l'applicazione viene eseguita.

  • VBL (Visual Basic License)
    Licenza di un controllo utente ActiveX, necessaria affinchè possa essere utilizzato il componente.

  • EXE (Executable / Executable ActiveX)
    Programma standard o ActiveX compilato in maniera direttamente utilizzabile dal computer.

  • DLL (DLL / DLL ActiveX)
    Libreria DLL standard o ActiveX compilata, utilizzabile da parte di altri programmi.

  • VBX (Visual Basic eXtension)
    Le estensioni di Visual Basic, utilizzabili in altri programmi e nelle versioni precedenti alla 5 di Visual Basic. Sono molto simili ai controlli ActiveX.

  • OCX (User Control ActiveX)
    Controllo utente compilato, riutilizzabile all'interno di un altro progetto.

  • OCA (User Control ActiveX Map)
    File di cache di un controllo utente compilato, utilizzato da Visual Basic per accedere più rapidamente ai controlli esterni.

  • TLB (Type Library)
    File di libreria dei tipi. Contengono le descrizioni di tipi di dati, moduli e interfacce di oggetti da esporre con la tecnologia ActiveX.

  • EXP (Binary Exportation Table)
    Tabella di esportazione di una DLL, contenente i simboli e i loro indirizzi per il riutilizzo di una DLL in altri linguaggi.

  • LIB (Binary Compiled Library)
    Tabella di importazione ed esportazione compilata per l'utilizzo di una DLL in altri linguaggi.

  • OBJ (Object compiled files)
    File compilato in linguaggio intermedio. Lo stesso sarà poi collegato al programma dal linker.

  • DEF (Definition Exportation)
    Elenco delle funzioni da rendere esportabili per il loro sfruttamento attraverso l'API.
