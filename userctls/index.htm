Controlli utente

In questa sezione sono presentati alcuni controlli utente sviluppati dal gruppo VB Simple.
Abbiamo preferito presentare dei controlli utente con estensione .CTL invece che degli OCX per dare al programmatore possibilità di personalizzazione e per evitare la fastidiosissima operazione di registrazione necessaria agli OCX in quanto componenti ActiveX.

Alcuni di questi controlli saranno utilizzati in alcuni progetti presenti all'interno del sito. Questa sezione è stata sviluppata appositamente per semplificare la spiegazione di alcuni articoli concentrandosi sul resto del codice anziché di alcune funzioni che saranno invece svolte dai nostri controlli utente.

Attualmente sono presenti i seguenti controlli utente:

  • FBI Horizontal Separator
    Una semplicissima linea separatrice orizzontale     - 27/05/2001

  • FBI Hyperlink
    Effettuare collegamenti ipertestuali     - 27/05/2001

  • FBI Shape Progress Bar
    Barra di avanzamento con Shapes     - 27/05/2001

  • FBI Change Resolution
    Cambio della risoluzione video     - 07/01/2002

  • FBI IP Address
    Immissione di un indirizzo IP     - 27/01/2002

  • FBI Graph Progress Bar
    Barra di avanzamento grafica     - 06/03/2002

  • FBI Edit FlexGrid
    Editare il contenuto di una FlexGrid     - 05/09/2002

  • FBI Select Window
    Seleziona una finestra mediante un mirino     - 06/10/2002

  • FBI Properties Browser
    Elenca le Proprietà contenute in un insieme Properties     - 14/10/2002

  • FBI Tray Icon
    Aggiunge un'icona nella System Tray     - 14/12/2002

  • FBI Spy Memory
    Visualizza il contenuto di un'area di memoria     - 18/12/2002

  • FBI Splitter & Sizer
    Separatore automatico per un gruppo di controlli     - 30/01/2003

  • FBI Shell ListView
    Visualizzare il contenuto di una cartella     - 02/11/2003
