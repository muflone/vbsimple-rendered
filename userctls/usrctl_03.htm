Il controllo FBI Shape Progress Bar è un'alternativa alla ProgressBardei Windows Common Controls.
Consiste in una barra di avanzamento costruita mediante l'utilizzo delle Shapesutile per mostrare all'utente l'avanzamento di un'operazione o elaborazione.

Proprietà

Utilizzabile in fase di progettazione
Accessibile in lettura in fase di esecuzione
Accessibile in scrittura in fase di esecuzione
  • Alignment As AlignmentConstants
    Restituisce ed imposta l'allineamento del testo.
    Determina la posizione del testo all'interno della barra colorata utilizzando un valore dell'enumerazione AlignmentConstants.

  • BackColor As OLE_COLOR
    Restituisce o imposta il colore per la barra di sfondo.

  • BorderColor As OLE_COLOR
    Restituisce ed imposta il colore del bordo della barra di avanzamento.

  • BorderStyle As BorderStyleConstants
    Restituisce ed imposta lo stile del bordo della barra di avanzamento.

  • Caption As String
    Restituisce il testo contenuto nella barra di avanzamento.

  • Decimali As Integer
    Restituisce ed imposta il numero di cifre decimali da visualizzare nella proprietà Caption.

  • Font As StdFont
    Restituisce ed imposta il tipo di carattere utilizzato per la Caption della barra di avanzamento.

  • ForeColor As OLE_COLOR
    Restituisce ed imposta il colore della barra principale.

  • Max As Double
    Restituisce ed imposta il valore massimo per la barra di avanzamento.

  • Min As Double
    Restituisce ed imposta il valore minimo per la barra di avanzamento.

  • MouseIcon As StdPicture
    Restituisce ed imposta un puntatore personalizzato per la barra di avanzamento.

  • MousePointer As MousePointerConstants
    Restituisce ed imposta il tipo di puntatore visualizzato sulla barra di avanzamento.

  • TextColor As OLE_COLOR
    Restituisce ed imposta il colore del testo della barra di avanzamento.

  • Value As Double
    Restituisce ed imposta il valore corrente della barra di avanzamento.

Metodi

  • Percentuale
    Tipo di metodo: Function
    Parametri richiesti: CifreDecimali As Integer (opzionale)
    Definizione: Function Percentuale([CifreDecimali As Integer]) As Double
    Restituisce il valore percentuale della barra di avanzamento.

Eventi

  • Change
    Parametri richiesti: (nessuno)
    Definizione: Event Change()
    Evento generato ad ogni cambio della proprietà Value.

  • Minimo
    Parametri richiesti: (nessuno)
    Definizione: Event Minimo()
    Evento generato quando la proprietà Value raggiunge il valore Min.

  • Massimo
    Parametri richiesti: (nessuno)
    Definizione: Event Massimo()
    Evento generato quando la proprietà Value raggiunge il valore Max.

  • Errore
    Parametri richiesti: ByRef MostraErrore As Boolean
    Definizione: Event Errore(MostraErrore As Boolean)
    Evento generato ad ogni impostazione errata.
