  • ACK
    Abbreviazione di Acknowledgment ovvero riconoscimento. Indica un'informazione inviata quale conferma di un dato ricevuto. Può indicare anche una certa comunicazione inviata per informare che una certà attività è ancora in corso.
    Vedi anche la dimostrazione del Nagle Algorithm.

  • ActiveX
    Tecnologia di progettazione di proprietà Microsoft, per lo sviluppo di componenti riutilizzabili in altre applicazioni. Sono detti anche Server In Process (DLL) e Server Out Of Process o Cross Process (EXE). I componenti ActiveX sono facilmente riutilizzabili all'interno di progetti Visual Basic.
    Essi si suddividono in:
    • DLL ActiveX, contenenti funzioni e classi istanziabili.
    • EXE ActiveX, vere applicazioni eseguibili che possono essere richiamate all'interno di altri programmi.
    • Documenti ActiveX, applicazioni utilizzabili in pagine web su Internet o in una rete Intranet.
    • Controlli OCX ActiveX, controlli con i quali progettare forms, del tutto simili ai controlli standard (Thunder) utilizzati nei normali progetti Visual Basic. Questi sono l'esempio più tradizionale di componenti ActiveX.
    È anche possibile creare nuovi ActiveX attraverso Visual Basic.
    Per inserire Controlli OCX ActiveX in un progetto selezionare la voce Componenti dal menu Progetto di Visual Basic. Se si tratta invece di DLL ActiveX selezionare la voce Riferimenti dal medesimo menu.

  • ActiveX Data Objects (ADO)
    Una delle tre interfacce più diffuse per l'accesso a fonti dati (database). È l'interfaccia di default per accedere a fonti dati tramite l'utilizzo del controllo ADODataConnector e rappresenta l'ultima tecnologia di accesso a fonti dati di qualunque tipologia mediante l'utilizzo di ODBC e di Provider specifici.
    Vedi anche Data Access Object (DAO).

  • Active Server Pages (ASP)
    Un ambiente di scripting server-side ovvero eseguito tramite un server web sul computer che ospita il server web stesso. Mediante la combinazione di HTML e componenti ActiveX è in grado di produrre pagine web statiche generate in funzione di elaborazioni in tempo reale.
    Viene utilizzato per generare pagine HTML o documenti XML in modo tale da comunicare con altri applicativi.
    Per un esempio vedi la Richiesta dedicata alla lettura di un database remoto.

  • ADO
    Acronimo di ActiveX Data Objects.

  • Alias
    Letteralmente "altrimenti detto" ed indica una sostituzione di nome di un dato. Tipicamente utilizzato nelle dichiarazioni API per assegnare un altro nome ad una funzione.
    Vedi anche mapping e Informazioni aggiuntive sui nomi di file in Windows.

  • Allocare
    Preparare e mettere da parte un'area di memoria per utilizzarla in varie operazioni. Un'area di memoria allocata é a disposizione del programma per memorizzare dati in essa.
    All'uscita del programma o del blocco dove l'allocazione é fatta, é necessario deallocare la stessa area, per renderla nuovamente disponibile agli altri programmi.

  • Ambiente di sviluppo
    Vedi IDE.

  • ANSI
    Acronimo di American National Standards Institute. Definisce il set di caratteri utilizzato in maniera standard in Windows 9x. Ogni carattere del set è costituito da un byte (8 bit). È pertanto un set di caratteri a byte singolo e permette fino a 256 simboli diversi.
    Vedi anche Unicode e OEM.

  • API
    Acronimo di Application Programming Interface.
    Consiste nell'informare il compilatore mediante un'istruzione apposita che si vuole accedere ad una funzione esportata di una libreria non ActiveX.
    Per effettuare una chiamata API si deve provvedere la dichiarazione completa della funzione che si desidera chiamare. Fatto questo si potrà utilizzare il nome della funzione esterna come se fosse una qualsiasi istruzione del linguaggio.
    Tuttavia le funzioni esterne non verranno incluse nel progetto in fase di linkage, ma il programma, ogni volta che avrà necessità della funzione suddetta, farà una chiamata esterna alla libreria in cui essa è contenuta.

  • Appunti
    Area di memoria comune utilizzata per contenere i dati delle operazioni Taglia, Copia ed Incolla. I dati al suo interno possono essere di vario genere e di più tipologie contemporaneamente, ma possono contenere solo un dato.

  • Array
    Gruppo di variabili, detti anche vettore, dello stesso tipo in memoria organizzate in sequenza, identificate dallo stesso nome ma distinte da un indice. Possono essere unidimensionali o multidimensionali e talvolta anche ridimensionabili.
    Sono trattati sempre come puntatori e pertanto sono sempre passati per riferimento.
    Vedi anche Collection.

  • ASCII
    Acronimo di American Standard Code for Information Interchange e definisce il set di caratteri più utilizzato nei personal computer.

  • Asincrono
    Modalità di esecuzione contemporanea ad un altro processo. Una chiamata ad un processo in modalità asincrona ad esempio indica che l'esecuzione del processo esterno non interromperà l'esecuzione del processo chiamante ovvero il processo che effettua la chiamata non attenderà la fine del processo chiamato per continuare la sua esecuzione.
    È opposto a Sincrono.

  • ASP
    Acronimo di Active Server Pages.
