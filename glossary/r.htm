  • Radio Button
    Nome originale del controllo Thunder OptionButton
    Per tanti anni il nome di questo controllo è stato Pulsante Radio, ma improvvisamente è stato deciso di cambiare il suo nome.

  • Recordset
    Un insieme di record estratti da una fonte dati (database), tipicamente mediante l'uso di una query SQL.
    Contiene al suo interno righe (record) e colonne (campi) per ognuna delle righe del set di record.
    L'oggetto è implementato nelle interfacce Data Access Object, ActiveX Data Objects (ADO) e Remote Data Service (RDS).
    Vedi anche Interfaccia.

  • Registro di Windows
    File residente in memoria dentro il quale sono memorizzati tutti i CLSID ed i nomi interni di tutti i componenti del sistema.
    Quasi tutti i programmi sfruttano il registro per scrivere dati di piccole dimensioni, in alternativa ai files di configurazione, per salvare i loro settaggi.
    Tutti i componenti ActiveX devono essere registrati per poter essere utiilzzati.

  • Remoto
    Indica che un componente hardware e software in una struttura di rete si trova installato in un computer diverso dal richiedente, in genere al di fuori dell'organizzazione in cui si trova il computer richiedente oppure su Internet.
    È opposto a locale.

  • RFC
    Acronimo di Request For Comments ed indica ognuno dei documenti che definiscono i protocolli standard di comunicazione sulle reti.
    Il collegamento a due archivi RFC è presente nella sezione Siti di altro genere.
    Vedi anche protocollo.

  • RGB
    Acronimo di Red Green Blue ovvero rosso, verde e blu. Consiste nella tripletta di colori la cui combinazione forma tutti i colori che un computer è in grado di rappresentare.
    Vedi anche l'HowTo dedicato alla gestione dei colori RGB.

  • Risorse
    Vedi File di risorse.

  • Routine
    Segmento di codice separato dal flusso principale del programma. Le routine sono alla base della programmazione procedurale, ma sono presenti anche nella programmazione ad oggetti e si dividono in funzioni e sub (dette a volte anche procedure).
    Una routine è richiamabile da alcuni punti del programma. Il richiamo di una routine provoca l'esecuzione delle istruzioni all'interno d'essa.
