  • Task
    Termine generico per indicare un'azione, un'operazione da eseguire; un programma in esecuzione in memoria è un task; può essere sinonimo di processo.
    Vedi anche Multitasking.

  • TCP/IP
    Acronimo di Transport Control Protocol / Internet Protocol.
    Protocollo di rete sul quale è basato l'intero funzionamento di Internet; comprende una serie di protocolli di applicazione.
    Vedi anche protocollo, IP, ICMP e RFC.

  • Thread
    Un segmento di codice che viene eseguito all'interno dello spazio di memorizzazione di un processo. All'interno di un processo possono essere in esecuzione un numero variabile di threads, ognuno dei quali svolgerà una funzione indipendente dagli altri threads.
    Vedi anche Processo e l'HowTo dedicato alla creazione di threads in VB.

  • Thunder
    Nome interno dei controlli standard presenti in Visual Basic, automaticamente collegati all'interno del programma compilato.
    I controlli Thunder sono 20:

    • Label
      Controllo per inserire informazioni non modificabili direttamente dall'utente.
    • CommandButton
      Pulsante standard per eseguire ogni genere di operazione.
    • TextBox
      Casella di testo a riga singola o multipla per permettere all'utente l'inserimento di dati.
    • Frame
      Contenitore con cornice per raggruppare per tipologia oggetti di qualunque genere.
    • PictureBox
      Contenitore progettato per inserire immagini, ma anche altri controlli.
    • CheckBox
      Casella con due stati selezionabili. Utile per scelte di genere logico (SI/NO).
    • OptionButton
      Pulsante di opzione (chiamato anche Pulsante Radio) che permette all'utente di attivarne solo uno in una serie. All'attivazione di un pulsante tutti gli altri si disattivano.
    • ComboBox
      Casella di testo a discesa. Utile per inserire testo ed immagazzinarne altre per dare possibilità di selezionare un valore dalla lista
    • ListBox
      Casella con voci selezionabili ma non modificabili dall'utente.
    • HScrollBar
      Barra di scorrimento orizzontale, utilizzata per spostare un valore
      numerico entro certi limiti.
    • VScrollBar
      Barra di scorrimento verticale, identica a quella orizzontale, ma questa è posta in posizione verticale.
    • Timer
      Cronometro programmabile. Allo scattare del tempo specificato un evento avverte l'utente. Utilizzato per compiere operazioni temporizzate quale l'aggiornamento di una grafica animata.
    • DriveListBox
      Casella a discesa con la lista dei dischi presenti nel sistema.
    • DirListBox
      Casella di selezione con la lista delle cartelle presenti nel percorso selezionato.
    • FileListBox
      Casella di selezione con la lista dei files rispondenti ad un certo criterio all'interno della cartella specificata.
    • Shape
      Permette di definire delle forme quali cerchi, ovali, rettangoli, etc. riempiti o vuoti.
    • Line
      Linea angolabile congiungente due punti. Utile per dare un tocco grafico o inserire delle indicazioni in un programma.
    • Image
      Controllo non contenitore per inserire immagini quali Bitmap da mostrare all'utente.
    • Data
      Controllo per gestire una tabella di un database in maniera semplice. Contiene anche tutti i comandi utili in un database.
    • OLE
      Controllo contenente un tipo di dati rispondenti al criterio dell'automazione OLE (Object Linking and Embedding), inseribili all'interno di altre applicazioni.

  • TLB
    Acronimo di Type Library.

  • Token
    Letteralmente "gettone", ogni occorrenza di un certo elemento nell'ambito di una ricerca o estrazione.

  • Trap
    Traduzione inglese del verbo intrappolare. Viene utilizzato per indicare l'intercettazione di una particolare situazione ad esempio la generazione di una situazione di errore da gestire.

  • Trascinamento
    Nel significato generico del termine indica l'operazione composta da:
    • la pressione di un pulsante del mouse su uno o più oggetti
    • lo spostamento del mouse senza il rilascio del pulsante premuto
    • il rilascio del pulsante del mouse
    Nel significato stretto del termine indica soltanto le prime due operazioni (drag) mentre l'ultima è detta rilascio (drop). L'operazione al completo prende il nome di drag'n'drop (trascina e rilascia).
    Vedi anche Trascinamento OLE.

  • Trascinamento OLE
    Un'operazione di trascinamento su oggetti di vario genere quali testi, immagini, files o altri elementi di tipo OLE.
    Vedi anche Trascinamento.

  • Type Library
    Letteralmente Liberia dei tipi e nel Component Object Model (COM) contiene le informazioni sui tipi di dati esposti negli oggetti ActiveX.
    Vedi anche Estensioni dei files di VB.
