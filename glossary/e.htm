  • Edizioni di Visual Basic
    Microsoft Visual Basic 5 viene distribuito in 5 diverse edizioni:
    • Visual Basic Working Model
      Versione base di Visual Basic, molto limitata, senza la possibilità di poter compilare il progetto sviluppato e con un massimo di 2 moduli per progetto.
    • Visual Basic Control Creation Edition (CCE)
      Versione limitata alla creazione e compilazione di Controlli OCX.
    • Visual Basic Learning Edition
      Versione limitata in alcuni aspetti. Permette lo sviluppo di progetti EXE Standard al puro scopo didattico. La licenza Microsoft non permette la distribuzione dei programmi compilati con questa edizione.
    • Visual Basic Professional Edition
      Prima edizione completamente funzionante. Con questa edizione è possibile sviluppare EXE Standard, EXE, DLL, componenti e documenti ActiveX.
      È inoltre possibile distribuire i programmi compilati con questa edizione.
    • Visual Basic Enterprise Edition
      Versione completa di VB. Include tutte le caratteristiche delle altre edizioni ed alcuni strumenti di controllo dei codici e funzionalità aggiuntive.

    Esistono inoltre 3 Service Pack per Visual Basic 5, che correggono molti errori, presenti in tutte le edizioni.

  • Enumerazione
    Tipo di dati numerico che permette di assegnare dei nomi costanti a dei valori raggruppati per tipo. Sarà possibile quindi utilizzare sia i numeri che i nomi dei valori enumerativi.
    Vedi anche la sezione Enumerazioni.

  • Esadecimale
    Sistema di numerazione a base 16. Le sue cifre vanno dallo 0 al 9 e dalla A alla F.
    poiché i dati in un computer sono memorizzati a byte, cioè a gruppi di 8 bit, si usa rappresentare i dati binari attraverso 2 cifre esadecimali, infatti 2 ^ 8 = 256 = 16 ^ 2.
    Vedi anche Sistemi di numerazione.

  • Estensione dei files
    Nei sistemi operativi MS-DOS® e Windows® tutti i files sono composti da due parti: il nome del file, fino ad un massimo di 8 caratteri e la sua estensione, massimo 3 caratteri, separate tra loro da un punto.
    Windows tramite uno strano trucchetto riesce a visualizzare fino a 255 caratteri per il nome e l'estensione, pur tuttavia memorizzando il file su disco nella forma classica (8 + 3).

    L'estensione di un file ne determina il tipo ed il programma associato quando l'utente clicca due volte sopra d'esso.
    È stabilito, per esempio, che quando si clicca due volte sopra un file con estensione BMP viene aperto il programma di grafica corrispondente, per esempio il Paint di Windows.
    Allo stesso modo i files con estensione VBP sono in realtà i progetti sviluppati con Visual Basic. Quando l'utente clicca due volte sopra uno di questi viene automaticamente aperto l'IDE di Visual Basic ed il progetto viene caricato.

  • Estensioni dei files di VB
    Come tutti i files anche quelli di Visual Basic hanno un'estensione. Abbiamo infatti le estensioni:
    • VBP - Visual Basic Project - Files di progetto.
    • VBG - Visual Basic Group - Gruppo di progetti.
    • VBW - Visual Basic Windows - Lista delle finestre aperte.
    • FRM - Form - Finestra.
    • FRX - Binary Form - Dati binari di un form.
    • CLS - Class - Modulo di classe.
    • BAS - Basic Module - Modulo.
    • CTL - User Control - Controllo Utente.
    • CTX - Binary User Control - Dati binari di un controllo utente.
    • PAG - Property Page - Pagina delleproprietà.
    • PGX - Binary Property Page - Dati binari di una pagina di proprietà.
    • DOB - User Document - Documento utente.
    • DOX - Binary User Document - Dati binari di un documento utente.
    • RC - Resources to Compile - File di risorse da compilare.
    • RES - Resources Compiled - File di risorse compilato.
    • DEP - Dependencies - Elenco delle dipendenze.
    • VBL - Visual Basic License - Licenza di un controllo utente ActiveX.
    • EXE - Executable / Executable ActiveX - Programma standard o ActiveX.
    • DLL - DLL / DLL ActiveX - Libreria DLL standard o ActiveX.
    • VBX - Visual Basic eXtension - Estensioni di Visual Basic.
    • OCX - User Control ActiveX - Controllo utente compilato.
    • OCA - User Control ActiveX Map - File di cache di un OCX.
    • TLB - Type Library - Libreria dei tipi.
    • EXP - Binary Exportation Table - Tabella di esportazione di una DLL.
    • LIB - Binary Compiled Library - Tabella di importazione ed esportazione.
    • OBJ - Object compiled file - File compilato in linguaggio intermedio.
    • DEF - Definition Exportation - Elenco delle funzioni da rendere esportabili.

  • Evento
    Momento legato all'attivazione di determinate azioni di un oggetto, come la pressione di un pulsante. Il programmatore può sfruttare l'attivazione di un evento per eseguire operazioni legate all'esecuzione dell'evento.
    Vedi anche Programmazione ad oggetti.

  • Extended Markup Language (XML)
    Il linguaggio di marcatura estendibile ovvero un linguaggio di marcatura come l'HTML ma che consente la definizione di nuovi tags per un proprio utilizzo. È molto utilizzato per scambiare dati di varia natura (ad esempio Database o in particolare Recordset) con semplicità in quanto esistono praticamente pochissime regole sintattiche da seguire ma le possibiltà di espansione sono infinite.
